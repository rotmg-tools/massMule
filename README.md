massMule
========

Scripts to manage mules.


# Perl scripts #


password.pl
-----------
Set random password for list of mules.


register.pl
-----------
Register mules.


requestVerify.pl
-----------
Request email verification for a list of mules.


verify.pl
-----------
Verify mules.


# Bash scripts #

md2txt.sh
-----------
Convert from muledump to plain text.

txt2md.sh
-----------
Convert from plain text to muledump.
